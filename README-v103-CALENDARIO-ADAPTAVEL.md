# v103 — Calendário adaptável

- Remove o limite fixo de cinco semanas do calendário dos clientes.
- Calcula automaticamente quantas semanas são necessárias para cada mês.
- Exibe calendários com quatro, cinco ou seis linhas, conforme o mês.
- Garante que todos os dias do mês apareçam na tela, inclusive os dias 30 e 31 em meses que ocupam seis semanas.
- A exportação em imagem usa a mesma quantidade dinâmica de linhas e inclui o mês completo.
- Atualiza o cache dos arquivos do navegador para carregar a correção imediatamente após a publicação.
